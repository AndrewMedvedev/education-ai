Техническое задание

Образовательный ИИ-агент на основе Telegram

Стек технологий: Сервис N8N + DeepSeek + База данных (supabase,) + RAG + Облако (Гугл, Яндекс)

Задачи:

1. При первом запуске ИИ-агент Telegram регистрирует в профиле + идентифицирует пользователя ID, Фамилия, Имя, Отчество, Группа, e-mail в Supabase. (УБЕДИТЬСЯ, ЧТО ПОЛЬЗОВАТЕЛЬ ЯВЛЯЕТСЯ ТЕМ, КТО ОН ЕСТЬ). ИИ-агент **записывает** пользователя (студента) на курс. Возможно: + одобрить заявку на курс. Валидация пользователя! Отсекать не нужных пользователей.
2. Тема по курсу (Например программирование на C# ) для изучения выбирается пользователем из **списка** через меню в Telegram. Список тем с описанием находится в таблице Supabase. ИИ DeepSeek генерирует учебный материал по выбранной теме. Шаблон учебного материала: (общая генерация информации DeepSeek) + ЕСЛИ ЕСТЬ ПО ТЕМЕ: ссылки с открытых источников + с видео хостинга + RAG (учебники с облака, презентации, видео ссылки с облако). Генерировать дополнительный учебный материал по выбранной теме: **Список** 10-15 **основных терминов** + **Контрольные вопросы 10-15 для самопроверки**.
3. (Если тема уже сдана студентом, то…)
4. Пользователь выбирает из подменю по теме **[ТЕСТ]** ИИ-агент генерирует тестовые вопросы в количестве 10-15 по теме для проверки усвоения теоретического материала (тест в виде собеседования вопрос от ИИ = ответ от пользователя) + ИИ-агент после тестирования выставляет итоговую отметку за тест по 100 бальной системе с подробными комментариями за каждый ответы пользователя и заносит **отметку** таблицу в БД Supabase.
5. Пользователь выбирает из подменю по теме **[Практическое задание]**. ИИ-агент генерирует практическое задание по теме. Сгенерированное задание заносит в таблицу БД Supabase.
6. Пользователь выполненное задание загружает на проверку из подменю по теме **[Проверка задания]** в виде: ссылка на репозиторий GitHub / ссылка на Облако / Файл в беседу
7. ИИ-агент анализирует (валидация) можно ли вообще проверить работу. Если невозможно, то сообщает («Невозможно проверить задание… некорректная ссылка или файл»), иначе проверяет по указанным критериям и выставляет по 100 бальной системе с подробными комментариями и замечаниями и заносит отметку в БД
8. Дополнительные возможности: ИИ-агент выдает через меню прогресс успеваемости (какие темы сданы, какие остались)
9. Дополнительные возможности: Пуш от ИИ агента уведомление о сроках сдачи задания.
